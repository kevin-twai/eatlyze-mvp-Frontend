# Eatlyze Frontend (Tech Minimal)
React + Vite + Tailwind 前端，連接你的 Render 後端。

## 開發
```bash
npm i
cp .env.example .env   # 設定 VITE_API_BASE_URL
npm run dev            # http://localhost:5173
```

## 建置
```bash
npm run build
npm run preview
```

## 環境變數
- `VITE_API_BASE_URL`：你的後端，例如 `https://eatlyze-backend.onrender.com`
